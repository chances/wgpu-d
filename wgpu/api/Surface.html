    Struct Surface

    A handle to a presentable surface.

    struct Surface ;

    A Surface represents a platform-specific surface (e.g. a window) to which rendered images may be presented. A Surface may be created with Surface.create.

    Fields

    NameTypeDescription
    id ulongHandle identifier.

    fromAndroid (androidNativeWindow) Create a new Surface from an Android handle.
    fromMetalLayer (layer) Create a new Surface from a Metal layer.
    fromWayland (surface, display) Create a new Surface from a Wayland window handle.
    fromWindowsHwnd (_hinstance, hwnd) Create a new Surface from a Windows window handle.
    fromXlib (display, window) Create a new Surface from a Xlib window handle.
